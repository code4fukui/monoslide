# monoslide

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

monoslide is an interactive web application for visualizing 8-bit numbers. It provides a simple and intuitive way to understand the relationship between binary, decimal, and hexadecimal number systems.

This project is a fork of [一日一創2012](https://fukuno.jig.jp/2012/monoslide).


![Screenshot of the monoslide application showing binary bits, a slider, and the corresponding hexadecimal and decimal values](./monoslide.png)


## Demo

Try it live: **https://code4fukui.github.io/monoslide/**

## Features

- **Multiple Representations:** View an 8-bit number (0-255) in binary, decimal, and hexadecimal formats simultaneously.
- **Interactive Slider:** Use the slider to quickly change the number.
- **Bit Toggling:** Click on any of the 8 bits to flip its value (0 or 1) and instantly see the result.
- **Keyboard Controls:** Use the left and right arrow keys to increment or decrement the number.

## Usage

To run the application locally, simply open the `index.html` file in your web browser.
