# monoslide

monoslideは、8ビットの数値を可視化するインタラクティブなWebアプリケーションです。2進数、10進数、16進数の関係をシンプルかつ直感的に理解できる方法を提供します。

このプロジェクトは[一日一創2012](https://fukuno.jig.jp/2012/monoslide)のフォークです。

![2進数のビット、スライダー、および対応する16進数と10進数の値を表示するmonoslideアプリケーションのスクリーンショット](./monoslide.png)

## デモ

ブラウザでお試しください: **https://github.com/code4fukui/monoslide

## 機能

- **複数の表現形式:** 8ビットの数値（0〜255）を2進数、10進数、16進数の形式で同時に表示します。
- **インタラクティブなスライダー:** スライダーを使って数値をすばやく変更できます。
- **ビットの切り替え:** 8つのビットのいずれかをクリックして値（0または1）を反転させ、その結果を即座に確認できます。
- **キーボード操作:** 左右の矢印キーを使って数値を増減できます。

## 使い方

アプリケーションをローカルで実行するには、`index.html`ファイルをWebブラウザで開くだけです。
